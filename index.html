¿Qué es Residential guardian system?

El país se puede considerar un país bastante inseguro y con alta tasa de mortalidad, personas no se sienten seguras a la hora de salir a la calle o aun estando en sus hogares por temor de un día regresar y no ver sus pertenecías como las tenía. Es por eso que decidimos crear un sistema para mayor seguridad en el país. El sistema de seguridad presentado tiene como fin hacer de una residencia una zona segura con un precio menor al que se encuentra en una residencia en una zona segura también de fácil uso para hacer el sistema más accesible contando con base de datos sistema para vigilantes administrador y también con cámaras de seguridad también cuenta con formularios de registro que comenzará a usar el sistema.
