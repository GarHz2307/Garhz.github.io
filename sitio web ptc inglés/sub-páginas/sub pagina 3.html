Who are we?

We are a team, who developed a security guard system, which want to make better the security of our country, El Salvador, and we want to guarantee the residents and visitors security inside a residenc; For the same sake, we decided to create Residential Guardian System.

Our motivation by which we decided to create Residential Guardian System is because nowadays the majority of residencies has no trustworthy secure ways to control. We create this system to facilitate the control of the residencies in El Salvador.
